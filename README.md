# test-process-quality

## Призначення репозиторію
Репозиторій є частиною дослідження з оцінювання якості процесу тестування програмного забезпечення та демонструє практичну реалізацію:

1. Модуль обчислення вектора пріоритетів матриці за методом Сааті.
2. Інтелектуальної системи пошуку дублікатів звітів про помилки.
3. Cистеми автоматичного формування оптимального розкладу виконання тестових сценаріїв між тестувальниками.
